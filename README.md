## 📸 Preview

> A dark-themed, responsive single-page application documenting every subject and unit across 6 semesters of B.Tech Data Science.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Semester Filtering** | Tab-based navigation to filter subjects by semester |
| **Live Search** | Instant search across all subjects and units (`⌘K` / `Ctrl+K`) |
| **Expand/Collapse** | Click any subject card to reveal unit details, or use "Expand All" |
| **Animated Counters** | Stats animate on scroll using IntersectionObserver |
| **Scroll Reveal** | Content gracefully fades in as you scroll down |
| **Python Skills** | Dedicated section showcasing Python ecosystem knowledge |
| **Back to Top** | Floating button appears after scrolling |
| **Fully Accessible** | ARIA labels, keyboard navigation, skip-to-content link, focus-visible styles |
| **Responsive** | Optimized for desktop, tablet, and mobile |
| **Print Friendly** | Clean print stylesheet — all units expanded, decorations removed |
| **SEO Ready** | Open Graph, Twitter Card, meta tags configured |

## 🚀 Quick Start

### View Live
Visit **[aryanmishra-dev.github.io/BTech-Subjects-](https://aryanmishra-dev.github.io/BTech-Subjects-/)** — no setup needed.

### Run Locally
```bash
# Clone the repo
git clone https://github.com/Aryanmishra-dev/BTech-Subjects-.git

# Open in browser
open index.html
# or
python -m http.server 8000   # then visit localhost:8000
```
