# B.Tech Data Science Academic Journey

A single-page, static website that showcases my B.Tech Data Science curriculum (RGPV Bhopal, 2023-2027), including semester-wise subjects and unit coverage.

## Preview

- Local preview: open `index.html` in a browser

## Purpose

This repository is intentionally minimal. It focuses on one polished HTML page that presents:

- Semester-wise academic progression
- Subject and unit breakdown
- Search/filter interactions for fast navigation

## Usage

1. Clone the repository.
2. Open `index.html` directly in your browser.

Optional local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
